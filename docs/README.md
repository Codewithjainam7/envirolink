## Overview

EnvironmentTech is a comprehensive waste management platform enabling:

- **Citizens** to report waste hotspots via images and GPS
- **Authorities** to track, assign, and resolve cleanup tasks
- **Field Workers** to receive and complete assignments
- **Public** to view transparency dashboards
